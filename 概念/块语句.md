{}

看上去是一个对象,但是块语句用的也是{}

{}在表达式里面或者赋值语句里面就是对象字面量

```js
var a = {};
({toString: function(){return 'hehe'}}) + '...' // hehe...
```

由于js中存在标签的语法
```js
    {
        name: function(){}
    }

    // 这样的代码往往会被解析成
    {// 代码块
        // 标签
        name: function() {} // syntaxError
        // 这就是为什么这样的语法经常会出错的原因
    }
```

但是当{}外层包上()的时候,内部的对象默认被识别为表达式而不是块语句,表达式形式的{}就是对象
